CSS Animations

Animation 1: Navigation/Header bar

In Header bar, I added a decorators container to contain moving decorating elements. Inside, I added 3 decorator elements, in which there is a bar element. Those bar elements are rectangular div elements that were rotated and had transparent background color. I animated those bar to move left and right by adding animation to the decorators to change their padding-left.

The animation is in every Header bar on every webpage. Link to the one on this page.

Animation 2: Banner title animation

In the banner on homepage, I added a banner-title element as the viewing box and banner-title-container to contain titles in different languages. Only the title that appear within the banner-title can be seen. The last title is the same as the first title. I animated the container to repeatedly move up 1 line-height and delay to make the title change. When it reaches to the last title, it immediately jump back to the first title and continue to move up, making the title looks like it always move up. The animation will pause when the user hover mouse over it.

Link to the animation on the banner on homepage.

Animation 3: "Back to top" arrow animation

At the bottom right corner of the page, I added a "back to top" button, which is an anchor element to the top of the page. The animation is nearly the same as the Banner title animation, and it plays once whenever the user is hovering on the button.

The animation is at the bottom right corner in every webpage. Link to the one on this page.
